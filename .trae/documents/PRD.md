# 1. 产品概述

一个完美复刻澳门赌场模式的骰宝（Sic Bo）游戏网页应用，提供真实的赌桌体验。
- 目标用户：对骰宝游戏感兴趣的玩家，希望体验澳门赌场真实氛围的用户
- 核心价值：通过精美的 UI 设计和完整的游戏机制，还原澳门赌场骰宝桌的真实体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 下注、查看结果、管理筹码 |

### 2.2 功能模块
1. **游戏主界面**：澳门赌场风格骰宝桌，完整还原赌桌布局
2. **下注系统**：支持多种下注类型（大小、单双、点数、组合、围骰等）
3. **骰子动画**：3D 骰盅摇骰动画，真实物理效果
4. **筹码管理**：虚拟筹码系统，不同面值筹码
5. **赔率结算**：自动计算赔率并结算
6. **历史记录**：显示最近开奖结果

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主页面 | 赌桌面板 | 完整还原澳门赌场骰宝桌布局，包含所有下注区域 |
| 游戏主页面 | 骰盅区域 | 3D 骰盅动画，摇骰后显示三颗骰子点数 |
| 游戏主页面 | 筹码栏 | 底部筹码选择区，1、5、10、25、100、500 面值筹码 |
| 游戏主页面 | 下注区域 | 各种下注选项的点击区域，显示已下注金额 |
| 游戏主页面 | 信息面板 | 显示余额、总下注、上局结果、赔率表 |
| 游戏主页面 | 历史记录 | 显示最近 20 期开奖结果 |

## 3. 核心流程

```
用户进入游戏 → 查看余额和赔率表
    ↓
选择筹码面值
    ↓
点击赌桌下注区域进行下注（可多次下注）
    ↓
点击"开骰"按钮
    ↓
骰盅动画播放（3D 摇骰效果）
    ↓
显示三颗骰子点数
    ↓
自动结算：计算中奖注项及赔率
    ↓
更新余额和显示结果
    ↓
可选择清除下注或继续下注
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深绿色赌桌背景（#0D5C2E），金色边框点缀
- **配色方案**：
  - 赌桌绿色：#0D5C2E, #1A7A3E
  - 金色：#D4A843, #F0D68A
  - 红色：#C41E3A (下注标记)
  - 白色：#F5F0E8 (文字)
  - 深色木质边框：#3E2723
- **按钮风格**：3D 立体按钮，金色边框，带有光泽效果
- **字体**：使用衬线字体（Playfair Display）模拟赌场风格，搭配无衬线字体（Lato）用于信息显示
- **布局风格**：桌面优先，完整还原赌桌布局

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 游戏主页面 | 赌桌 | 深绿色绒布质感背景，金色边框，下注区域网格布局 |
| 游戏主页面 | 骰盅区域 | 中央顶部，黑色骰盅，三颗白色骰子带点数 |
| 游戏主页面 | 下注区 | 所有下注选项以网格排列，标注中文名称和赔率 |
| 游戏主页面 | 筹码栏 | 底部横向排列，圆形筹码带面值数字 |
| 游戏主页面 | 信息面板 | 顶部或侧边，显示余额、下注总额、上局结果 |

### 4.3 下注类型及赔率
| 下注类型 | 说明 | 赔率 |
|----------|------|------|
| 大 | 点数 11-17（不包括围骰） | 1:1 |
| 小 | 点数 4-10（不包括围骰） | 1:1 |
| 单 | 点数总和为单数 | 1:1 |
| 双 | 点数总和为双数 | 1:1 |
| 指定点数 | 三颗骰子总和为指定数字 | 6-50:1 不等 |
| 双骰 | 指定两个骰子为相同数字 | 10:1 |
| 全骰（围骰） | 三颗骰子数字相同 | 24:1 |
| 指定围骰 | 指定数字的围骰 | 150:1 |
| 对子 | 两颗骰子数字相同 | 8:1 |
| 单一数字 | 指定数字出现在1/2/3颗骰子上 | 1:1 / 2:1 / 3:1 |

### 4.4 响应式设计
- 桌面优先设计，针对 1920x1080 及以上分辨率优化
- 支持 1366x768 以上分辨率
- 暂不支持移动端

## 5. 筹码系统
- 初始筹码：10000
- 筹码面值：1、5、10、25、100、500
- 每次下注需先选择筹码面值，再点击下注区域
- 可多次下注叠加
- 点击已下注区域可查看下注明细
